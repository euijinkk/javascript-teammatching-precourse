## 📌 구현할 기능 목록

- 탭 클릭에 따라 다른 메뉴 보여주기
- 크루 관리
  - 라디오 버튼 클릭 시 코스별 크루 관리 뜨기
  - 이름 입력받으면 크루 리스트에 추가
  - 삭제 버튼 클릭 시 confirm 띄우기
- 팀 관리
  - 코스별, 미션별 리스트 넣기
  - 확인 버튼 클릭 시
    - 1팀당 인원수 묻기
      - [예외] 1이상 자연수로 입력받기
      - [예외] 크루 인원 이하의 수로 입력받기
    - 크루 목록 뜨기
  - 팀 매칭 클릭 시
    - 팀 배치
    - 재매칭 버튼 생성
      - 재매칭 클릭시 팀 재배치
